=head1 NAME

soft-mad - Make the automated database from the /software tree.

=head1 DESCRIPTION

Creates the automated database file from the programs and files stored in
the /software tree.

B<NOTE:> This manual page only describes basic information for the program
titled soft-mad. If you are new to the SoftEnv system, refer to 'man
softenv-intro'.  For administration details, refer to 'man softenv-admin'.
More detailed information about this program can alo be found in those
manual pages.
